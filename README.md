# Burbujas
Proyecto realizado en JavaScript para la asignatura de Desarrollo Web en Entorno Cliente.

La aplicación consiste en un juego interactivo de burbujas dinámicas creado completamente mediante manipulación del DOM desde JavaScript. El usuario debe pulsar las burbujas antes de que desaparezcan para conseguir puntos.

Características principales del proyecto:

* Creación dinámica de todos los elementos HTML mediante JavaScript.
* Uso de eventos con addEventListener y función cross-browser compatible con navegadores antiguos.
* Generación automática de burbujas usando setInterval.
* Eliminación automática de elementos mediante setTimeout.
* Sistema de puntuación en tiempo real.
* Almacenamiento y recuperación del récord máximo usando localStorage.
* Uso de posiciones y colores aleatorios para las burbujas.
* Diseño dividido en panel de control y zona de juego mediante CSS.

Tecnologías utilizadas:

* HTML5
* CSS3
* JavaScript ES6
* DOM
* localStorage
* Temporizadores JavaScript

Proyecto orientado a practicar:

* Manipulación dinámica del DOM.
* Programación basada en eventos.
* Gestión de temporizadores.
* Persistencia de datos en navegador.
* Desarrollo de interfaces interactivas.
